## 🌐 Accéder au site

### En développement local

1. Ouvrez votre navigateur
2. Allez à **http://localhost:5173** ou **http://127.0.0.1:5173**
3. Vous verrez la page d'accueil

### Adresse IP locale

Si vous êtes sur un autre ordinateur du réseau:
- Remplacer `localhost` par l'adresse IP de la machine hébergeant le site
- Exemple: `http://192.168.1.100:5173`

### Menu de navigation

Depuis n'importe quelle page, vous trouvez:

## ✍️ S'inscrire

### Étape 1: Accès au formulaire d'adhésion

1. Ensuite la page d'accueil → **"S'enregistrer"** ou **"Adhésion"**
2. Ou cliquez directement: **Login** → **Pas encore de compte? → S'inscrire**

### Étape 2: Remplir le formulaire

| Champ | Description | Exemple |
|---|---|---|
| **Email** | Votre adresse email (unique) | jean@example.com |
| **Nom complet** | Votre prénom et nom | Jean Dupont |
| **Date de naissance** | JJ/MM/AAAA | 15/03/1990 |
| **Occupation** | Votre métier/fonction | Étudiant, Chercheur, Ingénieur |
| **Institution** | Université/Entreprise | Université Pierre et Marie Curie |
| **Niveau d'études** | BAC, Licence, Master, Doctorat | License |
| **Domaine** | Domaine de compétences | Informatique, Biologie, etc. |
| **Mot de passe** | Au moins 8 caractères | Générer un mot de passe sécurisé |
| **Motivation** | Pourquoi rejoindre BetaLab? | *Optionnel* |

### Étape 3: Soumettre

1. Relisez le formulaire
2. Cliquez sur **"S'inscrire"** ou **"Envoyer"**
3. ✅ Confirmé! Vous recevrez un email de confirmation

### Attendre la validation

- ⏳ Un administrateur valide votre compte (1-2 jours)
- 📧 Vous recevrez un email de validation
- ✅ Vous pouvez alors vous connecter

## 🔑 Se connecter

### Première connexion

1. Cliquez sur **"Connexion"** en haut à droite
2. Entrez:
   - **Email:** L'email utilisé à l'inscription
   - **Mot de passe:** Votre mot de passe
3. Cliquez sur **"Se connecter"**

### Après connexion

Vous verrez:
- Votre nom en haut à droite
- Un menu utilisateur (votre profil, déconnexion)
- Accès au tableau de bord

### Mot de passe oublié?

1. Sur la page de connexion → **"Mot de passe oublié?"**
2. Entrez votre email
3. Recevez un lien de réinitialisation
4. Créez un nouveau mot de passe
5. Reconnectez-vous

## 👤 Mon profil

### Voir mon profil

1. Menu → **"Mon profil"**
2. Vous voyez vos informations:
   - Données personnelles
   - Rôle (membre, chercheur, etc.)
   - Historique des activités
   - Nombre de contributions

### Modifier mon profil

1. Cliquez sur **"Modifier"**
2. Changez les champs souhaités
3. Cliquez sur **"Soumettre la modification"**
4. ⏳ L'admin approuvera votre modification

> **Note:** Les modifications ne sont visibles au public qu'après approbation admin

### Changer mon mot de passe

1. Menu → **"Paramètres"** → **"Changer mot de passe"**
2. Entrez:
   - Ancien mot de passe
   - Nouveau mot de passe (2x pour confirmer)
3. Cliquez sur **"Changer"**
4. ✅ Votre mot de passe est maintenant changé

## 🎯 Activités

### Voir toutes les activités

1. Menu → **"Activités"**
2. Vous voyez une liste des activités approuvées avec:
   - Nom et description
   - Statut (en attente, en cours, terminée)
   - Nombre de participants
   - Bouton "Participer"

### Filtrer les activités

- **Par statut:** Sélectionnez "En cours", "Terminée", etc.
- **Par domaine:** Si applicable
- **Par date:** Récents en premier

### Participer à une activité

1. Trouvez l'activité qui vous intéresse
2. Cliquez sur **"Participer"**
3. Confirmez votre participation
4. ✅ Vous êtes inscrit!

### Créer une nouvelle activité (si vous avez le rôle Chercheur)

1. Menu → **"Créer une activité"** ou Tableau de bord → **"Nouvelle activité"**
2. Remplissez le formulaire:
   - Nom de l'activité
   - Description détaillée
   - Domaine/Catégorie
   - Date et horaire
   - Lieu

3. Cliquez sur **"Créer"**
4. ⏳ L'équipe admin approuvera votre activité
5. Une fois approuvée, elle sera visible pour others

### Modifier une activité

1. Allez à **"Mes activités"**
2. Trouvez l'activité à modifier
3. Cliquez sur **"Modifier"**
4. Changez les détails
5. Cliquez sur **"Soumettre"**
6. ⏳ Admin approuvera

## 🎪 Événements

### Voir tous les événements

1. Menu → **"Événements"**
2. Liste complète des événements organisés par BetaLab
3. Chaque événement affiche:
   - Titre et description
   - Date et lieu
   - Type (conférence, atelier, hackathon, etc.)
   - Nombre de participants

### S'inscrire à un événement

1. Trouvez l'événement qui vous intéresse
2. Cliquez sur **"S'inscrire"**
3. Confirmez votre inscription
4. ⏳ Attendre l'approbation de l'équipe BetaLab
5. Une fois approuvé → Email de confirmation

### Mes inscriptions

1. Tableau de bord → **"Mes événements"**
2. Vous voyez vos inscriptions avec statut:
   - ⏳ En attente d'approbation
   - ✅ Approuvée
   - ❌ Refusée

### Se désinscrire d'un événement

1. **"Mes événements"** → Événement à quitter
2. Cliquez sur **"Se désinscrire"**
3. Confirmez
4. Vous êtes retiré de la liste

## ❓ Questions & Réponses

### Poser une question

1. Menu → **"Questions"** → Bouton **"Poser une question"**
2. Ou allez directement: **Menu → "Questions et réponses"**
3. Remplissez:
   - **Titre:** Votre question en une ligne
   - **Description:** Details et contexte
   - **Catégorie:** Sélectionnez un domaine
4. Cliquez sur **"Poser la question"**
5. ⏳ Modération par l'équipe (les questions approuvées seront visibles à tous)

### Voir toutes les questions

1. Menu → **"Questions"**
2. Vous voyez les questions approuvées
3. Informations affichées:
   - Question
   - Nombre de réponses
   - Auteur et date

### Lire une réponse

1. Cliquez sur la question
2. Lisez la description complète
3. Voyez la réponse de l'admin (si elle existe)
4. Consultez l'historique des réponses

### Mes questions

1. Tableau de bord → **"Mes questions"**
2. Vous voyez toutes vos questions:
   - Status: En attente/Approuvée/Rejetée
   - Nombre de réponses

## 💡 Suggestions

### Soumettre une suggestion

1. Menu → **"Suggestions"** (ou depuis dashboard)
2. Cliquez sur **"Faire une suggestion"**
3. Remplissez:
   - **Titre:** Courte description
   - **Description:** Détails et justification
   - **Note:** De 1 (pas utile) à 10 (très utile)
   - **Catégorie:** Sélectionnez le domaine
4. Cliquez sur **"Envoyer"**
5. ⏳ L'équipe approuvera (les suggérées approuvées sont publiques)

### Voir les suggestions publiques

1. Menu → **"Suggestions"**
2. Vous voyez les suggestions approuvées:
   - Titre et description
   - Note moyennes
   - Auteur et date
   - Nom du contact au cas de suivi

### Mes suggestions

1. Tableau de bord → **"Mes suggestions"**
2. Tous vos soumises:
   - Status (En attente/Approuvée/Rejetée)
   - Retours de l'équipe (si rejetée)

## 🔐 Paramètres & Sécurité

### Accéder aux paramètres

1. Menu utilisateur (coin supérieur droit) → **"Paramètres"**

### Paramètres disponibles

#### 1. Profil
- Modifier informations personnelles
- Ajouter/modifier photo de profil
- Changer email (nécessite confirmation)

#### 2. Mot de passe
- Changer mot de passe
- Recommendations: 12+ caractères avec majuscules/minuscules/chiffres

#### 3. Sécurité
- **Sessions actives:** Voir où vous êtes connecté
- **Déconnexion distante:** Terminer les autres sessions
- **Historique de connexion:** Voir les tentatives

#### 4. Confidentialité
- Qui peut voir votre profil
- Qui peut vous contacter
- Partage de données

#### 5. Notifications
- Email notifications: Activer/désactiver
- Quels emails recevoir (approvals, mises à jour, etc.)

### Bonnes pratiques de sécurité

✅ **À faire:**
- Change votre mot de passe tous les 3 mois
- Utiliser des mots de passe forts et uniques
- Vérifier la connexion en cas d'activité suspecte
- Déconnecter après chaque session

❌ **À ne pas faire:**
- Ne partager jamais votre mot de passe
- Ne cliquer pas sur les liens suspects
- Ne pas publier d'informations personnelles
- Ne pas utiliser votre mot de passe ailleurs

## 💬 Aide & FAQ

### Questions fréquentes

**Q: Combien de temps pour valider mon compte?**  
A: Généralement 1-2 jours ouvrables. Vérifiez vos spams si vous n'avez pas reçu l'email.

**Q: Je ne peux pas me connecter?**  
A: 
1. Vérifiez l'email et le mot de passe
2. Utilisez "Mot de passe oublié" pour réinitialiser
3. Vérifiez que votre compte a été validé par un admin
4. Essayez un autre navigateur

**Q: Comment participer à une activité?**  
A: Allez à "Activités", trouvez celle qui vous intéresse, et cliquez "Participer".

**Q: Mes modifications de profil n'apparaissent pas?**  
A: Les modifications doivent être approuvées par un admin. Attendez 1-2 jours.

**Q: Peur d'un bug ou d'une erreur?**  
A: Cliquez sur "Contact" et détaillez le problème. L'équipe technique investigators.

**Q: Comment nous faire part de suggestions d'amélioration?**  
A: Utilisez le formulaire "Suggestions" ou contactez-nous directement.

**Q: Puis-je exporter mes données?**  
A: Contactez l'équipe pour un export de vos données personnelles.
